About

Welcome. This is a personal blog where I write about machine learning, mathematics, and whatever else catches my attention.

The posts here tend toward the technical — theorems, proofs, algorithms — but I try to make the ideas accessible and to be honest about what we know versus what we assume.

This site is built with plain HTML and CSS, hosted on GitHub Pages. No frameworks, no build step, no tracking. Just text.
